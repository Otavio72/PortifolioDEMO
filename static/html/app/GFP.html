Gerenciador de Finanças Pessoais (GFP)

GFP é uma aplicação web full stack criada para simplificar o gerenciamento de contas pessoais, com visualização gráfica dos dados e extração automática de informações de boletos.

Sobre o projeto

O GFP permite o envio de boletos das empresas CPFL, Energisa, Naturgy e VIVO. Os dados desses boletos são extraídos via OCR, seguidos de uma filtragem com expressões regulares para capturar apenas o valor e a data do documento.

Visualização de Dados

Os dados processados pelo sistema são apresentados em três tipos de gráficos — pizza, radar e linhas — utilizando a biblioteca Chart.js, permitindo análise visual clara e interativa das informações coletadas.

OCR

O boleto é recortado, o OCR Tesseract extrai os textos, que são filtrados por regular expressions para capturar data e valor, e os dados finais são armazenados em SQLite.

Gráficos

Interface

A interface permite ao usuário selecionar a empresa fornecedora do boleto — CPFL, Naturgy, Energisa ou Vivo — e enviar o arquivo para processamento de forma rápida e intuitiva. Desenvolvida com HTML e CSS puros, prioriza simplicidade e usabilidade, garantindo que a interação seja direta, sem distrações, e que os dados sejam encaminhados corretamente para o sistema de OCR e armazenamento.

OCR Tesseract

Ao receber o boleto, a imagem é recortada, mantendo apenas a parte inferior relevante. Em seguida, o OCR Tesseract é aplicado para extrair todos os textos presentes, que são então filtrados por regular expressions para identificar especificamente a data e o valor do documento. Por fim, os dados extraídos são armazenados em um banco de dados SQLite para uso posterior.

Gráficos Chart.js

Os dados armazenados no banco são recuperados e visualizados por meio de três tipos de gráficos interativos — radar, pizza e linhas — utilizando a biblioteca Chart.js. Essa abordagem permite analisar diferentes aspectos das informações de forma clara: o gráfico de linhas acompanha tendências ao longo do tempo, o radar facilita comparações multidimensionais entre variáveis, e o gráfico de pizza oferece uma visão proporcional de componentes específicos. A visualização interativa contribui para que o usuário interprete rapidamente padrões e insights relevantes, tornando a análise de dados mais intuitiva e eficaz.

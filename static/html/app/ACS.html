Assetto Corsa Stints (ACS)

ACS surgiu durante minha participação em um campeonato da World Sim Series (WSS). Nos treinos, percebi a necessidade de uma análise mais detalhada dos stints (sequências de voltas), o que inspirou a criação deste projeto

Sobre o projeto

O ACS é uma aplicação local que extrai dados de telemetria do jogo Assetto Corsa, envia os tempos de volta para um servidor com banco de dados MySQL, e os recupera para gerar gráficos comparativos entre dois stints.

Engenheiro virtual

Esses dados são enviados à API do GEMINI, onde um "engenheiro virtual" interpreta os resultados e fornece feedback técnico via chat.

Datalogger

Roda dentro do jogo Assetto Corsa, coletando dados via API do jogo, acessada por shared memory (memória compartilhada).

Servidor

Recebe e guarda os dados no banco de dados, além de enviar feedback para a interface.

Interface

Recupera os dados do MySQL cria um grafico com 2 stints selecionados e envia os dados para o engenehrio virtual

Datalogger template de Hunter Vaners

Através da API acessada pelo SharedMemory, o datalogger coleta em tempo real os dados do jogo — como telemetria e tempos de volta — e os envia ao servidor para armazenamento e análise, ao mesmo tempo, atualiza a interface dentro do próprio jogo, fornecendo feedback imediato ao usuário.

Servidor

Recebe os dados do datalogger via socket TCP/IP, processa as informações e as armazena em um banco de dados MySQL para análise posterior. Além disso, gerencia a comunicação em tempo real com a interface do usuário, enviando notificações e atualizações que refletem os dados armazenados.

Interface engenheiro virtual

Recupera os dados armazenados no MySQL e gera gráficos comparativos entre dois stints (sequências de voltas em uma corrida) selecionados pelo usuário. Além disso, integra-se à API do Google Gemini, que atua como um engenheiro virtual: analisa os dados recebidos, fornece feedback em tempo real ao piloto e responde a dúvidas por meio de chat interativo.

Versão de Demonstração ACSvDEMO

Como o ACS depende da Shared Memory do Assetto Corsa para extrair dados em tempo real, sua execução requer o jogo instalado; sem ele, o sistema não funciona. Para contornar essa limitação e permitir testes sem o simulador, foi criada a versão ACSvDEMO, baseada em uma versão anterior do projeto, que utiliza arquivos .csv para simular os dados do jogo.
